# MangoI Video - 망고아이 화상영어 솔루션

WebRTC 기반 자체 화상영어 솔루션입니다.

## 주요 기능

- 화상 통화: WebRTC 기반 1:1 및 그룹 영상 통화
- - 판서 기능: 실시간 화이트보드
  - - 메신저: 실시간 채팅
    - - 녹화 기능: 수업 녹화 및 다운로드
      - - 수업 등록/관찰/데모방
       
        - ## 기술 스택
       
        - Next.js 16 + TypeScript + Tailwind CSS + WebRTC
       
        - ## 시작하기
       
        - ```
          npm install
          npm run dev
          ```
